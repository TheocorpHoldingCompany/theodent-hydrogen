# theodent
Shopify storefront for Theodent toothpaste

## Quick Start
- Node: >= 16.14
- Package manager: Yarn (recommended)

### 1) Configure environment
- Copy `.env.example` to `.env` and fill values:
  - `PUBLIC_STORE_DOMAIN`
  - `PUBLIC_STOREFRONT_API_TOKEN`
  - `PRIVATE_STOREFRONT_API_TOKEN` (if needed)
  - `PUBLIC_STOREFRONT_ID` (optional)

### 2) Install dependencies
- `yarn install`

### 3) Run the dev server
- `yarn dev`

### Other scripts
- Build: `yarn build`
- Preview: `yarn preview`
- Lint: `yarn lint`
- Tests: `yarn test`
